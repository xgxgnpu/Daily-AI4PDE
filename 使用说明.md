# AI4Physics ArXiv 每日论文追踪系统 — 使用说明

## 1. 项目概述

本项目自动从 [arXiv](https://arxiv.org/) 检索 AI for Physics 领域的最新论文，按主题分类整理，生成结构化的 Markdown 文档。

**数据来源**: arXiv API (通过 `arxiv` Python 库)
**代码链接**: GitHub Search API (搜索论文 ID 对应的开源仓库)

### 覆盖的研究方向

| 编号 | 类别 | 关键词示例 |
|------|------|-----------|
| 1 | Scientific Machine Learning & PINNs | PINNs, DeepONet, Neural Operators, SciML |
| 2 | Dynamical Systems & PDE Learning | PDE, Koopman, ROM, Surrogate Modeling |
| 3 | Quantum Physics & Information | QML, VQE, Tensor Networks, Schrödinger |
| 4 | Materials Science & Condensed Matter | MLIP, DFT, Crystal Structure, Force Fields |
| 5 | Fluid Dynamics & Plasma Physics | CFD, Navier-Stokes, Turbulence, MHD |
| 6 | Particle & High Energy Physics | HEP, Jet Tagging, Lattice QCD |
| 7 | Astrophysics & Cosmology | Dark Matter, Gravitational Waves, LIGO |
| 8 | Climate, Weather & Geophysics | Weather Forecasting, Seismic Inversion |
| 9 | Generative Models & Discovery | Diffusion Models, Symbolic Regression, UQ |

## 3. 配置说明 (`config.yaml`)

```yaml
# 基础设置
base_url: "https://arxiv.paperswithcode.com/api/v0/papers/"
user_name: "YuzeHao2023"          # GitHub 用户名
repo_name: "daily-arxiv-ai4physics"  # 仓库名

# 显示选项
show_authors: True   # 是否显示作者
show_links: True     # 是否显示链接
show_badge: True     # 是否显示徽章
max_results: 10      # 每个关键词返回的最大论文数

# 发布选项
publish_readme: True   # 更新 README.md
publish_gitpage: True  # 更新 GitHub Pages
publish_wechat: False  # 更新微信版本

# 文件路径
json_readme_path: './docs/ai4physics-arxiv-daily.json'
md_readme_path: 'README.md'

# 关键词配置 (可自定义)
keywords:
  "你的主题名称":
    filters: ["关键词1", "关键词2", "多词关键词"]
```

### 自定义关键词

编辑 `config.yaml` 中的 `keywords` 部分即可添加/修改/删除搜索方向。每个方向下的 `filters` 列表中的关键词会用 `OR` 连接进行搜索。

## 4. 本地运行

### 抓取最新论文

```bash
cd /root/autodl-tmp/daily-arxiv-ai4physics
python daily_arxiv.py
```

运行后会:
1. 按 `config.yaml` 中的关键词查询 arXiv
2. 更新 `docs/` 目录下的 JSON 数据文件
3. 生成/更新 `README.md` (Markdown 表格格式)
4. 生成/更新 `docs/index.md` (GitHub Pages 格式)

### 更新论文代码链接

```bash
python daily_arxiv.py --update_paper_links
```

此命令会遍历已有论文，通过 GitHub Search API 查找对应的开源代码仓库。

## 5. 每日自动更新

### 方式一: GitHub Actions (推荐，适合 fork 后使用)

项目已配置 `.github/workflows/ai4physics-arxiv-daily.yml`:
- **定时触发**: 每12小时自动运行一次 (UTC 0:00 和 12:00)
- **手动触发**: 在 GitHub 仓库 → Actions 页面 → "Run Arxiv Papers Daily" → "Run workflow"

fork 本仓库后，GitHub Actions 会自动运行，无需额外配置。

### 方式二: 本地 crontab 定时任务

```bash
# 编辑 crontab
crontab -e

# 添加以下行 (每天早上8点运行)
0 8 * * * cd /root/autodl-tmp/daily-arxiv-ai4physics && python daily_arxiv.py >> /tmp/arxiv_daily.log 2>&1
```

### 方式三: 使用 screen 后台运行 (AutoDL 环境)

```bash
# 创建一个循环脚本
screen -dmS arxiv_daily bash -c '
while true; do
    cd /root/autodl-tmp/daily-arxiv-ai4physics
    python daily_arxiv.py
    echo "$(date): 更新完成" >> /tmp/arxiv_daily.log
    sleep 43200  # 12小时 = 43200秒
done
'

# 查看运行状态
screen -r arxiv_daily
```

## 6. AI4PDE 专用脚本

如果只关注 PDE/PINN 方向，可使用专用脚本:

```bash
python ai4pde_today.py
```

功能:
- 专注检索 PINNs / Neural Operators / PDE 相关论文
- 返回标题、作者、**完整摘要**、arXiv 链接、代码链接
- 输出到终端并保存为 `ai4pde_today.md`
- 默认返回最新30篇

## 8. 常见问题

### Q: 搜索不到论文?
- 检查 `config.yaml` 中关键词是否正确
- arXiv API 有速率限制，过于频繁可能被暂时限制
- 检查网络连接是否正常

### Q: 如何只搜索特定方向?
- 编辑 `config.yaml`，只保留你感兴趣的 `keywords` 部分
- 或者使用 `ai4pde_today.py` 等专用脚本

### Q: GitHub Actions 没有运行?
- 确认仓库 Settings → Actions → General 中允许了 workflows
- fork 的仓库默认可能需要手动启用 Actions
- 检查 Actions 页面的运行日志

### Q: 如何增加返回论文数量?
- 修改 `config.yaml` 中的 `max_results` 值 (默认10)
