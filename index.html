Method Taxonomy
Neural Network Methods for Solving PDEs
├── PINN / KAN              → Daily-PINN
│   ├── Embed PDE residuals in loss (no labeled data)
│   └── Forward/inverse problems, parameter identification
│
├── ELM / Random Features   → Daily-ELM
│   ├── Single hidden layer + random fixed weights + least squares
│   └── Ultra-fast training, no backpropagation
│
├── Data-Driven Neural Ops  → Daily-DataDriven-NeuralOperator
│   ├── Learn operator mappings (function → function)
│   └── DeepONet, FNO, GNO, U-NO
│
├── Physics-Informed Neural Ops → Daily-PhysicsInformed-NeuralOperator
│   ├── Physics-constrained operator learning
│   └── PI-DeepONet, PINO, PIFNO
│

│   └── Adam, L-BFGS, Natural Gradient, Quasi-Newton
│
├── Singular PDEs           → Daily-SingularPDE
│   └── Corner/interface/boundary-layer singularities
│
└── High-Dimensional PDEs   → Daily-HighDimPDE
    └── Deep BSDE, Deep Ritz, Tensor Networks, Theory
